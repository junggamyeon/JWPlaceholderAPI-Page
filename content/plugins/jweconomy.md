### Supported Placeholders

Below is a list of placeholders provided by the JWEconomy expansion:

| Placeholder | Description |
|---|---|
| `%jweco_balance%` | Show default currencies with shortened format (e.g. 1.5M, 10k). |
| `%jweco_balance_raw%` | Show exact raw amount without shortening (e.g. 1500000.00). |
| `%jweco_balance_<currency>%` | Show balance for a specific custom currency. |
| `%jweco_balance_raw_<currency>%` | Show raw balance for a specific custom currency. |
